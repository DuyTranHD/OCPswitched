# OCPswitched

This folder gathers all codes, data, and results files, which include:

1, Filippov's one-time and 2-times exploited: simple-test.extensions and simple-test-2.extensions

2, Optimal control of a system of material points in a straight line with dry friction: dryfriction1fixopt.extensions

3, Optimal control of a point mass on a rough plane: dryfrictionpmrpfixopt.extensions

Note that "extensions" stands for ".cpp" (C++ code file), ".dat" (data file), ".txt", and ".log" (result files).
